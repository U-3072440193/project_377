Описание проекта:

Проект представляет собой интерактивную доску для визуализации и управлением рабочим процессом.

Ориентировочно проект будет разбиваться на несколько элементов:
home - домашная страница
users — управление пользователями, профилями и аутентификацией, личные кабинеты.
	В перспективе добаление функции ведения переписок между пользователями через личный кабинет.
	
boards — модели досок и колонок.
	Интреактивная модель на основе React без перезагрузки страницы. С возможностью перетаскивания задач
	от доски к доске.Доска содежит колонки, колонки содержат задачи. Задачи с возможностью комментирования.
	Цветовые индикаторы по приоритету.
tasks — задачи, комментарии, вложения и статус.
	Комментирование с загрузкой и хранением файлов.
	В перспективе добавление сторонних API с нейросетями(Подсказки при комментировании,автоматическое создание задач из текста).
